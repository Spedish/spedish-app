# Spedish Webapp
## Dependency
- NodeJS  
```
brew install node
```
- MongoDB 
```
brew install mongodb
```
- GruntCLI  
```
npm install -g grunt-cli
```
- Bower  
```
npm install -g bower
```

## Build
```
npm install
```

## Run
1. Start MongoDB
```
mongod
```
2. Start server
```
node index.js
```
